# Task 1 - Data Analysis using Pandas and Matplotlib

## Project Description
This project loads a CSV dataset using Pandas and performs basic data analysis tasks such as:
- Displaying dataset information
- Checking missing values
- Calculating average of a numeric column
- Generating visualizations (Bar chart, Scatter plot, Heatmap)

## Folder Structure
```
Task1_Data_Analysis/
│── data/
│     └── dataset.csv
│── analysis.py
│── requirements.txt
│── README.md
│── output/
│     ├── bar_chart.png
│     ├── scatter_plot.png
│     ├── heatmap.png
```

## Installation
```bash
pip install -r requirements.txt
```

## Run the Project
```bash
python analysis.py
```

## Visualizations Included
- Bar Chart (Top 10 values frequency)
- Scatter Plot (between two numeric columns)
- Correlation Heatmap

## Insights
- The script calculates average values of numeric columns
- Heatmap shows correlation between numeric features
- Scatter plot visualizes relationship between columns
